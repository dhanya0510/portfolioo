This project is a personal portfolio website designed to showcase my skills, projects, certifications, and contact information. It serves as a professional platform for recruiters, faculty members, and peers to explore my work and technical competencies.

The website is developed using HTML, CSS, and JavaScript, ensuring a modern, responsive, and interactive user experience.

Features

Responsive Design: Works on desktop, tablet, and mobile devices.

About Me Section: Displays personal introduction, skills, and achievements.

Projects Section: Highlights academic and personal projects with descriptions and links.

Certificates Section: Showcases relevant certifications and accomplishments.

Contact Form: Allows visitors to send messages directly through the website.

Downloadable CV: Provides an option to download the resume in PDF format.

Smooth Animations: Interactive elements with CSS transitions and JavaScript effects.
